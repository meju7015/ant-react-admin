## React + NextJS + ANT Admin

Reactjs : https://ko.reactjs.org/ \
Nextjs : https://nextjs.org/ \
Ant Design : https://ant.design/ 

리액트 프레임워크 Nextjs + ANT Components 라이브러리를 이용한 빠른 시작 백엔드 프로젝트.

## 🔍 프로젝트 설치
```bash
# 의존성 설치
$ yarn install

# 빌드
$ yarn build

# 개발 서버
$ yarn dev

# 서버 시작
$ yarn start
```
